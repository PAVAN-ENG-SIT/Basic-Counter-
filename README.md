# Counter App 

A simple **Counter App** built with **HTML, CSS, and JavaScript**.  
This app allows users to **increment**, **decrement**, and **reset** a counter value with a simple and intuitive interface.

Open `index.html` in your browser to use the app.

## Features

- Increment the counter using the `+` button
- Decrement the counter using the `-` button
- Reset the counter to `0` using the `Reset` button
- Real-time display update
- Simple and responsive design

---

## Technologies Used

- **HTML** - Structure
- **CSS** - Styling
- **JavaScript** - Logic and DOM updates
